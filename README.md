# Software Lab Company Platform

A premium, conversion-optimized agency platform built with Next.js, TypeScript, and MongoDB.

## Documentation

- [Product Requirements Document (PRD)](docs/PRD.md)
- [Technical Architecture](docs/ARCHITECTURE.md)
- [Project Roadmap](docs/ROADMAP.md)

## Tech Stack

- **Frontend**: Next.js 16 (App Router), TailwindCSS 4, Framer Motion
- **Backend**: Next.js Route Handlers, MongoDB (Mongoose)
- **Auth**: NextAuth.js
- **UI**: Radix UI / shadcn/ui

## Getting Started

1. Install dependencies: `npm install`
2. Set up environment variables: `cp .env.local.example .env.local`
3. Run development server: `npm run dev`
